##Campo Minado:
  a lógica do campo minado é uma questão de perspectiva baseada na forma de um quadrado 3x3.
  O quadradinho central está cercado por outros 8 quadradinhos, formando assim o quadrado 3x3.
  Quando um quadradinho é revelado, podemos encontrar um campo vazio, um número ou uma bomba.
  O primeiro clique de cada partida é sempre uma questão de sorte, mas a partir do momento que temos um número, devemos usar a lógica.
